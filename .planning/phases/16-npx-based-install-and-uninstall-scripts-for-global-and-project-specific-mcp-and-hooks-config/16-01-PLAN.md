Create an npx-runnable CLI that installs/uninstalls AgentChat hooks and MCP server config for both global (~/.claude/) and project-level setups.

Purpose: Users can run `npx agent-chat install --global` or `npx agent-chat install` to configure Claude Code integration without manually editing JSON files. Uninstall cleanly removes only AgentChat entries.

Output: Working `bin/cli.js` CLI, refactored `merge-settings.cjs` with exported functions, updated `package.json` with `bin` field, and expanded integration tests.

The following functions exist in scripts/lib/merge-settings.cjs and need `module.exports`:
- `mergeSetup(settings, agentChatDir, projectDir)` — merges hooks + MCP into settings object
- `mergeTeardown(settings)` — removes AgentChat entries from settings object
- `readSettingsFile(filePath)` — reads JSON file, returns {} on error
- `writeSettingsFile(filePath, data)` — writes JSON with mkdir -p
- `buildHookEntry(eventType)` — creates hook config for an event
- `buildMcpEntry(agentChatDir, projectDir)` — creates MCP server config
- `hasAgentChatHook(hookGroups)` — checks if AgentChat hook exists in array
- `HOOK_EVENTS` — ['SessionStart', 'SessionEnd', 'PreToolUse', 'PostToolUse']
- `AGENT_CHAT_MARKER` — 'localhost:5555/api/hooks'
- `AGENT_CHAT_MCP_KEY` — 'agent-chat'

Global file locations:
- Hooks: ~/.claude/settings.json → hooks object
- MCP: ~/.claude/.mcp.json → mcpServers object

Project file locations:
- Hooks: <project>/.claude/settings.json → hooks object
- MCP: <project>/.mcp.json → mcpServers object

MCP entry format (global — no AGENT_CHAT_CWD):
```json
{
  "mcpServers": {
    "agent-chat": {
      "command": "node",

        "AGENT_CHAT_DB_PATH": "/Users/username/.agent-chat/data.db",
        "AGENT_CHAT_TENANT_ID": "auto",
        "AGENT_CHAT_AGENT_NAME": "claude-agent"
      }
    }
  }
}
```

       function mergeHooksOnly(settings) {
         const result = JSON.parse(JSON.stringify(settings));
         if (!result.hooks) result.hooks = {};
         for (const event of HOOK_EVENTS) {
           if (!Array.isArray(result.hooks[event])) {
             result.hooks[event] = [];

         return result;
       }

         if (!result.mcpServers) result.mcpServers = {};
         result.mcpServers[AGENT_CHAT_MCP_KEY] = mcpEntry;
         return result;
       }

    4. Add a new function `teardownHooksOnly(settings)` that removes ONLY hooks from a settings object:
       ```js
       function teardownHooksOnly(settings) {
         const result = JSON.parse(JSON.stringify(settings));
         if (result.hooks) {
           for (const event of Object.keys(result.hooks)) {
             if (Array.isArray(result.hooks[event])) {
               result.hooks[event] = result.hooks[event].filter((group) => {
                 if (!group || !Array.isArray(group.hooks)) return true;
                 return !group.hooks.some(
                   (h) => typeof h.command === 'string' && h.command.includes(AGENT_CHAT_MARKER)
                 );
               });
               if (result.hooks[event].length === 0) delete result.hooks[event];
             }
           }
           if (Object.keys(result.hooks).length === 0) delete result.hooks;
         }
         return result;
       }
       ```

         const result = { command: null, global: false, project: null };
         for (let i = 0; i < args.length; i++) {
           const arg = args[i];
           if (arg === 'install' || arg === 'uninstall') {
             result.command = arg;
           } else if (arg === '--global' || arg === '-g') {
             result.global = true;

             result.project = args[++i] || null;
